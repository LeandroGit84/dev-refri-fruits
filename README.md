# 🍹 Dev Refri Fruits

Projeto front-end inspirado em uma interface moderna de bebidas/refrescos com frutas, focado em prática de HTML, CSS e JavaScript.

## 🚀 Sobre o projeto

O **Dev Refri Fruits** é um projeto desenvolvido com o objetivo de treinar conceitos fundamentais de desenvolvimento web, criando uma interface visual atrativa baseada em bebidas de frutas.

A proposta é simular uma página interativa, com foco em:

* Manipulação do DOM
* Interatividade com JavaScript
* Estilização com CSS
* Organização de componentes visuais

## 🧠 Conceitos praticados

* Estruturação com HTML semântico
* Estilização com CSS (cores, layout e responsividade)
* Eventos no JavaScript (`onclick`, `addEventListener`)
* Manipulação dinâmica de elementos (`innerHTML`, `style`, etc.)
* Lógica condicional (if/else)

## 🎨 Funcionalidades

* Troca de conteúdo dinâmico na tela
* Alteração de estilos via JavaScript
* Interação com botões
* Simulação de UI de produto (refri/frutas)

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Estrutura do projeto

```
📁 dev-refri-fruits
 ├── index.html
 ├── style.css
 └── script.js
```

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/LeandroGit84/dev-refri-fruits.git
```

2. Acesse a pasta:

```bash
cd dev-refri-fruits
```

3. Abra o arquivo `index.html` no navegador

Pronto! 🍹
